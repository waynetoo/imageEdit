# 编辑图片Demo
编辑图片Demo是一个适用于 Android 平台的仿qq发送消息前编辑图片，帮助开发者在 Android 平台上快速编辑本地图片并保存。

###支持缩放，涂鸦，橡皮，箭头和文字。另可扩展涂鸦圆形，矩形，椭圆等。
###支持撤销和恢复操作。
